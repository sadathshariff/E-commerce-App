# ShopYard

ShopYard is an E-Commerce website,for branded and high quality clothes

## Technologies used

- HTML
- CSS

## Features

- Home page
- Product Listing Page
- Cart Management
- Wishlist Management
- Authentication
  - Sign-up
  - Login

## Project Link

- [ShopYard](https://shopyard.netlify.app/)
